# AgriSchoolCo
**A functional website ensuring active participation of all people interested in the agricultural sector. link is https://kavutsss.github.io/Direct-to-school-Agri-/**
## ABOUT THE PROJECT
AgriSchoolCo is a website designed to set success for the agricultural sector in our country.
Solving problems such as insufficient funds and resources for farmers, it is a website that will give people the technical know-how of what exactly and how exactly to improve the agricultural sector.
Our platform provides information on various projects we have completed, not forgetting a way in which one is able to register.
 
 ## FEATURES
 * **A resource library, containing curated materials that can be accessed by the users**
 * **Four webpages with access to our Home, About, and Contact pages**
 * **Live workshops and tutorials inclusive of the resource library**
 * **Access to registration into the platform**
 * **Links to our social media platforms**
